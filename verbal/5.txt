Вопрос: какой размер у структуры struct{}{}?

"Пустая" структура, как и её экземпляры, занимают 0 байт в памяти. Это удобно 
использовать в качестве типа канала, который не передаёт никаких данных, а лишь
сигнализирует о событии. Проверить это можно с помощью функции unsafe.Sizeof.

func main() {
    fmt.Println(unsafe.Sizeof(struct{}{})) // 0
}
