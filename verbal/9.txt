Вопрос: Сколько существует способов задать переменную типа slice или map?

В Go возможно задать переменную типа slice возможно тремя способами:
 - с помощью функции make
 - с помощью литеральной инициализации (slice := []int{1, 2, 3})
 - с помощью среза с существующего массива (slice := array[i:n])

Переменную же типа map можно создать двумя способами:
 - с помощью функции make
 - с помощью литеральной инициализации (map := map[string]int{"key": 1})
